Relative, Absolute, and Fixed Positioning

02/13/15

Relative

Setting an element to relative positioning will not affect the element until changes are made to its positioning attributes: top, bottom, left, right. Once a change is made to a position attribute, such as left: 20px;, the element will move 20 pixels to the right from where it would have normally been. A relatively positioned element is still part of the page flow.

Absolute

With absolute positioning, when the positioning attributes are set, the element will be positioned in relation to the first parent element that doesn't have static positioning. If there is none, then the element is positioned relative to (the page itself). An abolutely positioned element is not part of the page flow, meaning it doesn't affect other elements and isn't affected by other elements.

Fixed

A fixed position element will stay where it is in the browser when the window is scrolled up or down. This element is out of the page flow.
