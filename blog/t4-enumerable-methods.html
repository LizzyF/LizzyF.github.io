Map

12/16/14

An enumerable method applies to collections of data, such as arrays and hashes. The enumerable method goes through each element in the collection, making whatever changes to each that is specified in the code. An example would be:

names = ["leonard", "bernadette", "amy"] names.map {|scientists| scientists.capitalize}

In the example, there is an array called names. What the map method does is go through each element in the array, and capitalizes the first letter of each name, outputting the information to a new array called scientists. With the map method, the new array given will have the same number of elements as the beginning array.
