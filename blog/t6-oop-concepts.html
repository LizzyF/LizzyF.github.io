Variable Scope

1/9/15

Local variables are the most common variables in Ruby. There are no special symbols or syntax to signify a local variable. An example would be: name = "Joe". These variables can only be used in the method they are defined in.

Instance variables start with the @ symbol. These variables exist as long as the object it is associated with exists. Instance variables keep track of the state of the object.

Class variables begin with @@. These variables are available to all instances of the class, and are created at the class level.

Global variables begin with $. These variables are available throughout the application, but are used sparingly as these can cause problems later on.

Constants are signified by capitalizing all letters of the variable's name. These variables store information that never changes. If one tries to change the value assigned to the variable, Ruby will throw up a warning message.
